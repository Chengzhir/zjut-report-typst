## ZJUT-Report-Typst
[English](./README.md)

浙江工业大学一些实验报告的 Typst 模板.

> 注意： 非官方

- `template/template.typ` 模板文件
- `example.typ` 样例文件
- [example.pdf](./example.pdf) 样例文件的pdf
- `img` 静态文件

## Typst
由于 Typst 是一个很新的东西，（2023年4月刚出0.1.0版本）
模板随时可能损坏。

## 本模板

本模板尚未实现完全。

如果您有任何想法请提issue 或 pr
